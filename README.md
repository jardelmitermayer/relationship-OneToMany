## Sobre o projeto

O projeto é desenvolvido a partir do curso <a href="https://www.udemy.com/course/laravelcompleto/">Laravel 5.8 Completo - O mais poderoso Framework PHP </a> com o intuito de aprender sobre o relacionamento Um pra Muitos(OneToMany) com o framework Laravel.

# :construction_worker: Executando

```bash
# Clone o Repositório
$ https://github.com/jardelmitermayer/relationship-OneToMany.git
```

```bash
# Baixe as dependendências
$ composer install
$ npm install
$ npm run dev
```

```bash
# Execute
$ php artisan serve
```

Acesse <http://localhost:8000> para ver o resultado.
